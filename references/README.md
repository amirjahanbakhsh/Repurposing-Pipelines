# References

This folder is for project-wide literature, standards, source-acquisition notes, and citation tracking.

Use it for:

- lists of papers, reports, standards, and datasets we need;
- notes extracted from references;
- source traceability tables;
- links and exact search terms.

Do not commit licensed PDFs, paid standards, or copyrighted documents to GitHub. Keep those files locally and commit only the citation, notes, and model-relevant extracted facts.

| File | Purpose |
| --- | --- |
| `extracted_notes/` | Short notes from papers and reports, with model-relevant facts only. |
| `literature_index.csv` | Structured list of papers and reports we have found or received. |
| `source_acquisition_guide.md` | Exact titles, identifiers, links, and search terms for standards and papers we still need to obtain. |
