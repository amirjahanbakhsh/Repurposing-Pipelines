# Write-Up Drafts

This folder contains report-style text for the project.

| File | Purpose |
| --- | --- |
| `methodology.md` | First draft of the methodology section. |
| `journal_paper_draft.md` | Journal-paper style manuscript draft for future submission. |

The writing should stay readable for non-programmers while still being technically defensible.

When a model layer changes, update `methodology.md` and, where relevant, `journal_paper_draft.md` in the same work cycle so the write-up stays live with the code.
