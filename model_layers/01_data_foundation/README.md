# 01 Data Foundation

NSTA data extraction, cleaned pipeline attributes, candidate ranking, and data-quality reports.

| File | Purpose |
| --- | --- |
| `nsta_pipeline_attributes.json` | Raw public NSTA pipeline attributes. |
| `nsta_pipeline_metadata.json` | NSTA field metadata. |
| `nsta_pipeline_attributes.csv` | Cleaned NSTA table used by scripts. |
| `data/raw/nsta_pipeline_linear.geojson` | Full public NSTA route geometry, tracked with Git LFS. |
| `nsta_pipeline_completeness.md` | Completeness check for key fields. |
| `nsta_candidate_ranked.csv` | Ranked model-ready hydrocarbon candidates. |
| `nsta_candidate_ranking.md` | Human-readable ranking report. |
| `data_extraction_validation.csv` | Validation check for extraction and candidate counts. |
