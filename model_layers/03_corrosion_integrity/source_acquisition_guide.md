# Source Acquisition Guide

Date: 2026-06-10

This file lists the paid, subscription, or library-access documents we should try to obtain for the pipeline repurposing foundation.

Use the exact titles and search terms below in:

- Heriot-Watt University Library;
- ScienceDirect;
- BSI Knowledge;
- ISO store;
- ASME store;
- API Webstore;
- DNV Veracity / Rules and Standards Explorer+.

## What To Do After Downloading

When any document is available, save a copy outside GitHub if it is licensed or copyrighted.

Then we should extract only:

- exact clauses or section numbers we are allowed to cite;
- model-relevant requirements;
- input parameters needed;
- calculation methods;
- limitations;
- whether the source supports screening, validation, or final engineering approval.

Do not upload paid standards or licensed PDFs to GitHub.
